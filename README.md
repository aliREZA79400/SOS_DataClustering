# Automatic Data Clustering using Nature-Inspired Symbiotic Organism Search Algorithm

This repository contains Python code that implements the **Symbiotic Organism Search (SOS)** algorithm for automatic data clustering. The SOS algorithm simulates the symbiotic interaction strategies adopted by organisms to survive and propagate in an ecosystem. Clustering is a popular data analysis and data mining technique, and this repository demonstrates how to use the SOS algorithm to solve clustering problems.

## Installation
1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/automatic-data-clustering.git
   cd automatic-data-clustering
   ```

2. Create enviromet and install the required Python packages

3. Change directory to ADCSOS/SOS

4. python3 main.py --help (for help)

5. python3 main.py 200 50 5 (this command runs 5 times sos algorithm on each dataset with population size :50 epoch size :200 )
