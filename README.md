**SOS Data Clustering**
=========================

**Overview**
------------
https://www.sciencedirect.com/science/article/abs/pii/S0950705118304647

This repository contains a collection of algorithms and techniques for clustering data The goal of this repository is to provide a comprehensive set of tools and resources for data clustering, including algorithms, examples, and visualizations.

**Algorithms**
--------------

The following clustering algorithms are implemented in this repository:

* SOS 
* PSO
* DE
* FPA
* MVO
* ABC
* CS

**Datasets**
------------

The repository includes several sample datasets for testing and evaluating the clustering algorithms:

* Iris dataset
* Wine dataset
* Breast Cancer Wisconsin dataset
* ....

**Usage**
-----

To use the clustering algorithms in this repository, follow these steps:

1. Clone the repository using `git clone https://github.com/aliREZA79400/SOS_DataClustering.git`
2. Install the required dependencies using `pip install -r requirements.txt`
3. move to algorithn folder like `cd SOS`
4. `python3 main.py args`

**Visualizations**
-----------------

Several visualization tools are provided to help understand the clustering results in utils/visualize.py
